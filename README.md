#  Glowly Beauty

Website e-commerce kecantikan premium yang dibangun menggunakan **Next.js 14 (App Router)**, **TypeScript**, dan **Tailwind CSS**.  
Data produk diambil secara real-time dari **DummyJSON API**.

---
